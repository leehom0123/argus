# 分享

迟早会想把一个批次发给同事看，又不想让对方专门注册一个账号。
平台支持两种分享方式。

## 公开链接

在批次或任务详情页点 **Share → Create public link**。会得到一个
形如：

```
https://your-host/p/b/abcd1234efgh
```

slug 是不透明的随机 12 字符，不可枚举。任何拿到链接的人都可以
只读浏览，无需注册、不能下载原始数据。同一菜单可以撤销链接。

公开链接绝不暴露：

- Reporter token 名字
- 用户或主机的内部 ID
- 登录邮箱

会暴露的：

- 该批次的任务表与指标
- 损失曲线和上传的图
- 把主机名脱敏后的资源利用面板

## 项目级访问

项目详情页的 **Members** tab，可以按邮箱添加注册用户并赋予角色：

| 角色 | 权限 |
| --- | --- |
| Viewer | 只读看项目下所有内容 |
| Contributor | 只读 + 创建项目级 reporter token |
| Owner | 上述 + 管理成员 + 改名 |

项目成员即使没有公开链接也能看到该项目下的批次，这是小型实验室
（每人都有账号）的合理模型。

## 置顶与星标

- **Pin（置顶）** —— 把批次加到仪表盘的 *Pinned* 行，仅对置顶人
  本人可见。
- **Star（星标）** —— 标记为重要；星标批次会绕开保留期，并出现在
  *Settings → Starred* 下。

## 嵌入

公开批次链接接受 `?embed=1`，去掉导航栏，方便嵌进 Confluence 页面
或内部 wiki：

```html
<iframe src="https://your-host/p/b/abcd1234efgh?embed=1"
        width="900" height="500" frameborder="0">
</iframe>
```

CSP 默认允许同源嵌入，再加上 `ARGUS_EMBED_ORIGINS`
（逗号分隔）里指定的主机。

## 分享 *做不到* 的事

- 它不会把事件推到任何地方 —— 只读视图。
- 它不允许外部用户上报事件。只有 API token 能。
- 批次被软删除后链接立刻 404，不会苟活。
